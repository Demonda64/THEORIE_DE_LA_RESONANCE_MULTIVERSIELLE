# THEORIE_DE_LA_RESONANCE_MULTIVERSIELLE

Bibliotheque complete de la TRM (Theorie de la Resonance Multiversielle) avec corpus doctrinal, pipeline editorial et interfaces de lecture.

## Objectif du dossier

Ce dossier centralise:

- les textes canoniques et doctrinaux de la TRM;
- les supports de redaction et de revision;
- les versions de lecture web locales et deployables;
- les scripts utilitaires de generation et d'exploitation.

## Structure principale

- `02_STORY_BIBLE/`: noyau doctrinal (canon, traite, glossaire, textes maitres).
- `06_CHAPTERS/` a `09_REVISION/`: chaine de production narrative.
- `docs/`: site public/statique de consultation.
- `12_READER_SITE/`: interface locale de lecture/exploration.
- `TOOLS/`: scripts d'automatisation du pipeline.

## Usage autorise

Usage autorise uniquement dans les cas suivants:

- consultation personnelle ou interne de travail;
- edition par les personnes explicitement autorisees par le titulaire des droits;
- usage technique necessaire au fonctionnement du projet (indexation, previsualisation, export interne).
