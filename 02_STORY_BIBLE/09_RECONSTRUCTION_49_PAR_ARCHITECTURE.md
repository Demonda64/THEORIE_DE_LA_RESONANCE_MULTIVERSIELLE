# Reconstruction des 49 principes par architecture

## Statut
Architecture conceptuelle gelee. Ce document n'en modifie aucun element.

## Regle appliquee
Pour chaque principe :
1. Domaine principal (Ontologie, Dynamique, Phenomelogie, Epistemologie)
2. Objet principal (Reseau, Branche, Conscience, Noyau Identitaire, Temporalite Locale)
3. Statut logique (Loi fondamentale, Corollaire, Theoreme, Definition, Methode)
4. Deductibilite (Oui/Non) et base de deduction
